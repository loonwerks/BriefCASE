# agree\_to\_json
Translator of Eclipse AADL/AGREE model to JSON.
This depends on commons/util.  There is no UI handler.
To see how to use it, have a look at TA2/filter_checker.


## owner
Thomas `@tlogan`

## build
Currently, the easiest way to build this is to open this project and commons/util in the [osate-develop environment](http://osate.org/setup-development.html).
Then run a new instance of osate.
